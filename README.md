# Tecnologia Sem Medo 50+ - MVP

MVP estatico da area do aluno para validar a proposta comercial das trilhas:

- Tecnologia Sem Medo 50+: Curso Completo
- Computador Sem Medo 50+
- Celular Sem Medo 50+

## Como abrir

Abra o arquivo `index.html` no navegador.

## Acesso de demonstracao

- E-mail: `aluno@demo.com`
- Senha: `123456`

## O que este MVP faz

- Cadastro de aluno
- Login com e-mail e senha
- Escolha do plano no cadastro
- Liberacao das aulas conforme o plano escolhido
- Painel com progresso
- Checklist de aulas concluidas
- Links de materiais uteis
- Botao de suporte por WhatsApp

## Materiais pedagogicos

- `docs/plano-pedagogico-aula-atalhos.md`: modelo completo de uma aula da trilha Computador Sem Medo 50+.

## Observacoes importantes

Este MVP usa `localStorage`, ou seja, os dados ficam salvos apenas no navegador usado pelo aluno. Isso e suficiente para demonstracao e validacao da experiencia.

Para vender oficialmente, o proximo passo recomendado e criar uma versao com backend, banco de dados, recuperacao de senha, area administrativa e integracao de pagamento.
