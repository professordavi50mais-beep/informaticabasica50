# Plano pedagogico - Aula completa

## Curso

Tecnologia Sem Medo 50+

## Trilha

Computador Sem Medo 50+

## Aula

Atalhos de teclado essenciais

## Duracao sugerida

2 horas

## Publico-alvo

Pessoas 50+ que ja tiveram um primeiro contato com o computador, mouse e teclado, mas ainda dependem muito do mouse para realizar tarefas simples.

## Objetivo da aula

Ao final da aula, o aluno devera ser capaz de usar atalhos basicos de teclado para copiar, colar, recortar, desfazer, selecionar, salvar, imprimir, localizar palavras e alternar entre janelas abertas.

## Resultado esperado

O aluno conseguira realizar tarefas simples com mais autonomia e menos dependencia do mouse, ganhando confianca para escrever textos, navegar na internet e manipular arquivos.

## Materiais necessarios

- Computador ou notebook para cada aluno, se possivel.
- Teclado fisico.
- Mouse.
- Editor de texto simples, como Bloco de Notas, WordPad, Word ou Google Docs.
- Navegador de internet.
- Arquivo de texto simples para pratica.
- Folha impressa com os atalhos em letras grandes.

## Preparacao antes da aula

- Abrir um editor de texto nos computadores.
- Deixar um pequeno texto de pratica disponivel.
- Testar se os teclados estao funcionando.
- Conferir se os alunos sabem localizar as teclas `Ctrl`, `Alt`, `Windows`, `C`, `V`, `X`, `Z`, `A`, `S`, `P` e `F`.
- Deixar duas janelas abertas para a pratica de alternancia: navegador e editor de texto.

## Estrutura da aula

### 1. Abertura e acolhimento - 10 minutos

Objetivo:
Apresentar a utilidade dos atalhos sem transformar a aula em memorizacao.

Condução:

- Explicar que atalhos sao combinacoes de teclas que economizam tempo.
- Reforcar que ninguem precisa decorar tudo no primeiro dia.
- Mostrar que a aula sera baseada em repeticao e pratica.

Mensagem principal:

> Atalho nao e obrigacao. E uma ferramenta para facilitar o dia a dia.

### 2. Revisao rapida do teclado - 10 minutos

Objetivo:
Garantir que todos localizem as teclas antes de iniciar a pratica.

Conteudo:

- Tecla `Ctrl`.
- Tecla `Alt`.
- Tecla `Windows`.
- Letras usadas nos atalhos.
- Como pressionar duas teclas ao mesmo tempo.

Pratica guiada:

- Pedir para os alunos localizarem a tecla `Ctrl`.
- Pedir para manterem `Ctrl` pressionado e depois apertarem uma letra.
- Reforcar que nao e necessario apertar com forca.

### 3. Demonstracao do professor - 20 minutos

Objetivo:
Mostrar os atalhos em funcionamento antes de pedir que os alunos pratiquem.

Atalhos demonstrados:

- `Ctrl + C`: copiar.
- `Ctrl + V`: colar.
- `Ctrl + X`: recortar.
- `Ctrl + Z`: desfazer.
- `Ctrl + A`: selecionar tudo.
- `Ctrl + S`: salvar.
- `Ctrl + P`: imprimir.
- `Ctrl + F`: localizar uma palavra.
- `Alt + Tab`: alternar entre janelas abertas.
- `Windows + D`: mostrar a area de trabalho.

Exemplo usado pelo professor:

1. Abrir um texto simples.
2. Selecionar uma frase.
3. Copiar a frase com `Ctrl + C`.
4. Colar a frase em outra linha com `Ctrl + V`.
5. Recortar uma palavra com `Ctrl + X`.
6. Desfazer a alteracao com `Ctrl + Z`.
7. Salvar com `Ctrl + S`.
8. Localizar uma palavra com `Ctrl + F`.
9. Alternar entre editor de texto e navegador com `Alt + Tab`.

### 4. Pratica assistida - 45 minutos

Objetivo:
Fazer cada aluno repetir os atalhos com acompanhamento.

Exercicio 1: copiar e colar

1. Abrir o texto de pratica.
2. Selecionar uma frase.
3. Pressionar `Ctrl + C`.
4. Clicar em uma nova linha.
5. Pressionar `Ctrl + V`.

Exercicio 2: recortar e desfazer

1. Selecionar uma palavra.
2. Pressionar `Ctrl + X`.
3. Observar que a palavra saiu do lugar.
4. Pressionar `Ctrl + Z`.
5. Observar que a palavra voltou.

Exercicio 3: selecionar tudo e salvar

1. Pressionar `Ctrl + A`.
2. Observar que todo o texto ficou selecionado.
3. Clicar fora para remover a selecao.
4. Pressionar `Ctrl + S`.
5. Salvar o arquivo com um nome simples.

Exercicio 4: localizar palavra

1. Abrir o texto ou uma pagina da internet.
2. Pressionar `Ctrl + F`.
3. Digitar uma palavra.
4. Observar onde a palavra aparece.

Exercicio 5: alternar janelas

1. Abrir o navegador e o editor de texto.
2. Pressionar `Alt + Tab`.
3. Alternar entre as janelas abertas.
4. Usar `Windows + D` para mostrar a area de trabalho.

## Intervalo curto

Duracao: 5 minutos, se a turma demonstrar cansaco.

Observacao:
Em turmas 50+, e melhor fazer uma pausa breve do que seguir com baixa atencao.

## Exercicio individual - 15 minutos

Objetivo:
Verificar se o aluno consegue executar uma sequencia sem ajuda imediata.

Tarefa:

1. Copiar uma frase.
2. Colar a frase abaixo.
3. Desfazer uma alteracao.
4. Salvar o arquivo.
5. Procurar uma palavra no texto.
6. Alternar para o navegador.

Critério de sucesso:

O aluno nao precisa fazer com velocidade. O sucesso e conseguir completar a sequencia entendendo o que cada atalho faz.

## Revisao e fechamento - 15 minutos

Objetivo:
Reforcar os atalhos mais importantes e tirar duvidas.

Perguntas de revisao:

- Qual atalho usamos para copiar?
- Qual atalho usamos para colar?
- Qual atalho ajuda quando fazemos algo errado?
- Qual atalho usamos para salvar?
- Qual atalho usamos para encontrar uma palavra?

Atalhos prioritarios para memorizar:

- `Ctrl + C`
- `Ctrl + V`
- `Ctrl + Z`
- `Ctrl + S`
- `Ctrl + F`

## Tarefa para casa

Pedir que o aluno pratique pelo menos uma vez antes da proxima aula:

1. Abrir um texto simples.
2. Copiar uma frase.
3. Colar a frase.
4. Desfazer uma alteracao.
5. Salvar o arquivo.

## Material de apoio sugerido

Criar uma folha impressa com:

- Nome do atalho.
- Combinacao de teclas.
- Para que serve.
- Exemplo pratico.

Exemplo:

| Atalho | Para que serve | Exemplo |
| --- | --- | --- |
| `Ctrl + C` | Copiar | Copiar uma frase |
| `Ctrl + V` | Colar | Colar a frase em outro lugar |
| `Ctrl + Z` | Desfazer | Corrigir uma acao feita por engano |
| `Ctrl + S` | Salvar | Salvar um documento |
| `Ctrl + F` | Localizar | Procurar uma palavra em um texto |

## Cuidados pedagogicos

- Nao apresentar atalhos demais de uma vez.
- Repetir os atalhos principais varias vezes.
- Evitar cobrar velocidade.
- Incentivar o aluno a olhar a folha de apoio.
- Corrigir com calma quando o aluno apertar as teclas na ordem errada.
- Reforcar que o mouse continua sendo valido; o atalho e uma alternativa.

## Observacoes para o professor

Esta aula funciona melhor quando o professor circula pela sala durante a pratica. Alguns alunos podem ter dificuldade motora para pressionar duas teclas ao mesmo tempo, entao vale demonstrar formas mais confortaveis de usar as duas maos.

Tambem e importante observar se os alunos confundem `Ctrl + C` com `Ctrl + V`. Quando isso acontecer, retomar o exemplo "copiar primeiro, colar depois".
